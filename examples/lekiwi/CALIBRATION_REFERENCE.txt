┌─────────────────────────────────────────────────────────────────────────┐
│ MOVEMENT CONTROLS                                                       │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   W - Forward (X+)              Q - Up (Z+)         R - Pitch Up        │
│   S - Backward (X-)             E - Down (Z-)       F - Pitch Down      │
│   A - Left (Y-)                                                         │
│   D - Right (Y+)                                                        │
│                                                                         │
│         W                          Q                     R              │
│       A   D                      E   E                 F   F            │
│         S                          E                     F              │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│ STEP SIZE CONTROL                                                       │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   + or =  →  Increase step (double)                                    │
│   -       →  Decrease step (half)                                      │
│                                                                         │
│   1  →  10.0 mm   (Coarse - positioning)                               │
│   2  →  1.0 mm    (Medium - approach)                                  │
│   3  →  0.1 mm    (Fine - calibration)                                 │
│                                                                         │
│   Workflow: 1 → 2 → 3  (coarse to fine)                                │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│ ACTIONS                                                                 │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   P      →  Print current position                                     │
│   T      →  Test draw (20mm square)                                    │
│   H      →  Home (return to safe position)                             │
│   SAVE   →  Save current position                                      │
│   DONE   →  Finish and save calibration                                │
│   QUIT   →  Cancel and exit                                            │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│ WORKSPACE LIMITS (Safe Operating Range)                                │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   X (Forward/Back):    0.15 to 0.25 m   (150 to 250 mm)                │
│   Y (Left/Right):     -0.10 to 0.10 m   (-100 to 100 mm)               │
│   Z (Up/Down):         0.08 to 0.15 m   (80 to 150 mm)                 │
│                                                                         │
│   Stay within these limits for reliable operation!                     │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│ Z_CONTACT CALIBRATION WORKFLOW                                         │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   1. Start:        Robot at X:0.175 Y:0.0 Z:0.15 (high up)             │
│                                                                         │
│   2. Position:     W W W (forward over paper)                           │
│                    D D (right if needed)                                │
│                                                                         │
│   3. Approach:     Press 2 (1mm steps)                                  │
│                    E E E E (lower slowly)                               │
│                                                                         │
│   4. Fine-tune:    Press 3 (0.1mm steps)                                │
│                    E E (lower until pen touches)                        │
│                                                                         │
│   5. Test:         T (test draw)                                        │
│                    Should see marks if touching                         │
│                                                                         │
│   6. Save:         DONE (save z_contact)                                │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│ COMMON ISSUES & SOLUTIONS                                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   "Position outside workspace"                                          │
│   → Press H to return home, or move opposite direction                 │
│                                                                         │
│   "IK failed"                                                           │
│   → Use smaller steps (press - or 3)                                    │
│   → Return home (H) and try again                                       │
│                                                                         │
│   Pen doesn't touch paper                                               │
│   → Z too high, lower more (E with step 3)                              │
│   → Test with T after adjusting                                         │
│                                                                         │
│   Pen scratches/drags                                                   │
│   → Z too low, raise slightly (Q with step 3)                           │
│                                                                         │
│   Lost/confused                                                         │
│   → Press P to see position                                             │
│   → Press H to return home                                              │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│ MENU OPTIONS (Main Screen)                                             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   1  →  Quick workspace test    (verify robot works)                   │
│   2  →  Find Z_CONTACT height   (main calibration)                     │
│   3  →  Calibrate page corners  (define drawing area)                  │
│   4  →  Free jog mode           (explore workspace)                    │
│   5  →  Test drawing            (verify calibration)                   │
│   6  →  Exit                    (return to safe & quit)                │
│                                                                         │
│   Recommended: 1 → 2 → 5 (test, calibrate, verify)                     │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│ PRO TIPS                                                                │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   • Always start with coarse steps (1), refine to fine (3)             │
│   • Use T frequently to test if pen touches                            │
│   • Position X/Y first, then adjust Z                                  │
│   • The position where you calibrate Z can be your page origin         │
│   • Save often in free jog mode (SAVE command)                         │
│   • Use H (home) if you get lost                                       │
│   • P (print) shows exact position in meters and mm                    │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
